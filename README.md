# Research Workflow Automation Using AutoHotkey
This project demonstrates how AutoHotkey can automate repetitive research tasks. The scripts are inspired by workflow improvements I developed while working in investment research operations, but have been recreated from scratch using generic examples. The goal is to reduce repetitive typing, improve consistency, and save time during data aggregation and creating new tasks.
# Why I built this
While working in research operations I noticed many repetitive actions that could be automated. Learning AutoHotkey introduced me to process automation and helped me think differently about improving productivity. This repository contains simplified examples that demonstrate those concepts.
